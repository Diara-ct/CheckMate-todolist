# Checkmate To-Do List

**Checkmate** est ma propre application de To-Do List que j’ai conçue entièrement de A à Z.  
J’ai travaillé sur le design, le logo, les pages et le codage.

---

## Mon travail
- **Direction artistique et logo** : j’ai créé un logo simple et élégant sur Illustrator.  
- **Design et UX/UI** : toutes les pages sont pensées pour être claires et faciles à utiliser. J’ai fait les maquettes et le prototypage sur Figma.  
- **Développement** : j’ai codé le projet en HTML, CSS et JavaScript.

---

## Pages du projet
Le projet a 5 pages principales :  
1. Page d'inscription 
2. Page de connexion  
3. Page d'accueil
4. Page de gestion et de stockage des listes 
5. Page pour ajouter des tâches.

Toutes sont **responsive** et adaptées à différents écrans.

---

## Fonctionnalités
- Ajouter, supprimer et marquer les tâches comme terminées  
- Design clair et moderne 
- Les tâches restent enregistrées même si on ferme la page (stockage local)

---

## Technologies
- HTML, CSS, JavaScript  
- Figma pour les maquettes  
- Illustrator pour le logo  

---

## Structure du projet
- index.html : page d’accueil
- connexion.html / inscription.html : accès utilisateur
- taches.html : affichage des tâches
- ajout.html : ajout d’une tâche
- main.js : logique JavaScript
- style.css : design global

Les données (email et tâches) sont sauvegardées en local avec localStorage.
